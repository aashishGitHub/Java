# Java Core Platform

## Java: Two Identities

### Programming Language
- Data types, arrays, OOP, classes
- *Course assumes solid language knowledge*

### Runtime Environment (JRE)
- **Focus of course**
- Also: **JRE** / **Java SE**
- Core platform

---

## Java Technologies

### Java FX
- Rich client apps
- Library of classes (uses JRE)

### Java EE / Jakarta EE
- Enterprise apps
- **Builds on JRE** (adds services)

### Android
- Google (separate development)
- Java language ✓ | Different runtime ✗
- Similar: Collections, core classes
- Different: Threading model, configuration

---

## Key Points

- JRE = Core platform for Java FX, Java EE, Jakarta EE
- Android = Separate but benefits from JRE knowledge

### .NET Comparison
- JRE ≈ .NET Runtime/CLR
- Java SE ≈ .NET Framework Core
- Java EE ≈ .NET Enterprise features
- Java FX ≈ WPF/WinForms
